Episode II

WEB BUNDLER AWAKEN


In the vast expanse of the Quarkiverse, a new entity emerges - the Web Bundler. This Quarkus extension is not a destroyer, but a harmonizer, bringing a new perspective to full-stack web apps and components.

It offers zero-configuration bundling and minification for your web app scripts (js, ts), web-dependencies (React, Bootstrap, Lit, ...), and styles (css, scss) with source-map included.

Web Bundler does not seek to overthrow Quinoa, the trusted ally of many a NodeJedi developers. Instead, it presents a compelling case for those who seek a different path closer to the backend force.

So, if you're a developer standing at the crossroads, wondering whether to choose Quinoa or Web Bundler, remember this - both have their strengths, and both can help you create awesome web apps.

The choice, as always, is yours.
